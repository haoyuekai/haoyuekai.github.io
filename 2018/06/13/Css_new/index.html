新奇好玩的CSS

发表于 2018-06-13 | 分类于 Css

最近在大漠的博客中看到好很多有意思的css效果，今天稍微整理一下。只做基本效果展示。

内凹角效果

大体思路，伪类元素:before定位到父元素顶点处，利用box-shadow属性填充元素除:before外的其他区域，在 :before处形成内凹角。

上诉效果基本上已经实现了内凹角的效果 ，为了使用方便，还可以使用css自定义属性，控制内凹角的半径和位置，形成一个简单的组件。

在深入一点，可以加入js控制内凹角的半径，利用js改变自定义变量的值。

文字渐变

这部分效果主要是使用background-clip:text、color:transparent和linear-gradient来完成。
